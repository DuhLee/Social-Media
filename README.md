## 프로젝트 소개

<p align="justify">
[개인프로젝트] 소셜미디어 웹사이트
  
포스트를 올려서 다른 사람들과 공유할 수 있는 사이트

로그인하여 자신의 포스트를 올리거나 다른 사람의 포스트에 좋아요를 누를 수 있습니다.
포스트 상세페이지에서 상세 내용과 제목과 태그와 관련 있는 포스트를 추천해 줍니다.
</p>
